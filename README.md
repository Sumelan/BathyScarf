# BathyScarf / 罰襟巻
Configuration files for my Acer Laptop.

## Details
- **OS**: NixOS
- **Desktop Environment**: Hyprland
- **Theme**: catppuccin macchiato
- **Bar and Widget**: Eww
- **Terminal Emulator**: Kitty
- **Shell**: Zsh with Oh_my_zsh
- **Notifier**: dunst
- **Launcher**: rofi
- **TUI File Manager**: Yazi
- **GUI File Manager**: Thunar
